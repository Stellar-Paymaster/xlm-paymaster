    Interface WhiteLabelTokens

    White-label color token system for enterprise partners.

    Partners supply a WhiteLabelTokens object; the system validates it, generates a scoped CSS block, and applies it to a container element so the standard Tailwind/CSS-variable-driven components pick up the brand colors automatically.
